# imposibility-cli

Generate your own domain name

## How To Use
### Installing with npm
```bash
npm install -g imposibility-cli
```

### Installing with yarn
```bash
yarn add global imposibility-cli
```

after installed you just run

```bash
imposibility-cli
```
